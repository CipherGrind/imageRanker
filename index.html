Image Ranker

1. Choose images to rank

( Accepts most image types like: PNG, JPG, GIF, WEBP )
( total size limit is 100MB )

2. Click to preview images

3. Change ratings of the images

4. Click to reorder images

5. Edit image names

( optional )

6. Enter album folder name

( you will use this when saving the images )

7. Click to update image names

8. Click to save images

( this will generate a zip folder to save as )
( please wait a few seconds to minutes for processing )
